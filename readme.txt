== Description ==
BBS e-Popup support features to produce and manage popups.<br />
You can set popups for websites conveniently and efficiently.<br />
BBS e-Popup support shortcode for the posts or pages.<br />
This is a free word press popup plug-in that can generate popups infinitely. (For PCs)<br />
Include 25 templates and 36 close buttons.
  
<h4>* 한국어 : </h4>

<strong>[Main Features] </strong>

Generate popups easily using the editor for desired contents.<br />
Use images, HTML or other codes to add popups, and it is the same as adding word press contents pages.<br />
By using condition functions for specific users, it offers the functions to use popup in the exact location. to expose to specific users or users that access a specific page.<br />
Possible to manage use of popups generated infinitely.<br />
Select popup animation to set the popup method.<br />
Provides functions to set exact location and font size. With the fixed popup in the center, you can set the exact location in the website.<br />
You can change image, background, border, shadow, popup period message and types.<br />
You can change the word press site and harmonious popup shape and feel.

== Installation ==
1. Install BBS e-Popup either via the WordPress.org plugin directory, or by uploading the files to your server.<br />
2. After activating BBS e-Popup, connect to WordPress.com to enable the BBS e-Popup features.<br />
3. Offers various languages such as English, Japanese, Chinese and Korean.<br />
3. That's it.  You're ready to go!
